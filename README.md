# SonarSample

![CI](https://github.com/mohamedelghachi/SonarSample/actions/workflows/ci.yml/badge.svg)

## Description

Ce projet est un exemple d'application Python avec intégration SonarQube pour l'analyse de qualité de code.

## Fonctionnalités

- Fonction d'addition simple
- Vérification de palindromes
- Tests unitaires avec unittest
- Intégration continue avec GitHub Actions
- Analyse de qualité de code avec SonarQube

## Installation

```bash
pip install -r requirements.txt
```

## Tests

Pour exécuter les tests :

```bash
python -m pytest tests/
```

ou

```bash
python tests/test_app.py
```
